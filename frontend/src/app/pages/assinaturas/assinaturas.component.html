Assinaturas pessoais

Um card para cada alias SMTP encontrado na sua conta Microsoft 365.

Outlook desktop (Windows)

Selecione as assinaturas desejadas, marque a padrão e baixe o instalador. Após baixar, execute o arquivo .bat com duplo clique. Reinicie o Outlook após a instalação.

Outlook Web

No Outlook Web a assinatura é configurada manualmente. Use o botão Copiar para Outlook Web em cada card e siga as instruções abaixo.

@if (instrucoesOwaAbertas()) {
